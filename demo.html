Smart Answer Evaluator

I developed a deep learning model capable of classifying students' answers into three categories: partial correct, correct, and incorrect, by comparing them with an ideal answer. Leveraging BERT architecture as a foundation, I further customized it to accommodate manual input or image uploads for answers. The application calculates marks using probabilities obtained from the model, applying a formula I developed specifically for this purpose. Users can also select the type of checking—either hard checking or soft checking—according to their preference. This flexibility allows them to adjust how rigorously the system evaluates the results based on their specific needs or criteria.

Technologies Used: Transformers, Tesnorflow, FastAPI, Flask, HTML/CSS

VLC Media Controller Using Hand Gestures

In this project, I developed a deep learning model to classify hand gestures, enabling users to control the VLC media player seamlessly. The model recognizes gestures such as fist (pause/play), OK sign (confirm), two fingers up (volume up), palm (volume down), and other gestures as commands for actions like skipping tracks. This integration allows users to interact with VLC Media Player intuitively through hand gestures detected and classified by the deep learning model.I fine-tuned the VGG16 model using the HaGrid dataset, achieving an impressive accuracy of 96%.

Technologies Used: Tensorflow, TKinter, OpenCV
